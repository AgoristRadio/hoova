Hoova

The Bitcoin Wallet Sweeper Agent

Download .zip of Windows GUI View on GitHub

The Windows GUI.

The Windows GUI is under development and available to closed beta testers. Currently it supports automatic basic balance sweeping of private key to a bitcoin address. Wallet and BrainWallet sweeping are under development.

The Ruby Library

After we have the GUI ready to go, we will release the general library as a Ruby gem. This can be used to write your own apps, make cron jobs etc.

The Ruby Testing Application

The Github repo has the current code being developed for testing on the Bitcoin Testnet.
