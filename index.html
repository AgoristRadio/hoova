Hoova

The Bitcoin Wallet Sweeper Agent

What Does Hoova Do?

Hoova is a Bitcoin Agent that will automatically Sweep entire Balances from one location to another. Sweep the entire Bitcoin balance for a specific Bitcoin Address using its private key, or an entire Bitcoin Wallet, or a Brain Wallet, to any other Bitcoin Address you specify.

Current Features

  • Automation: Once a sweep is setup, the system will watch for payments and initiate the sweeping.
  • Security: When using your local Bitcoin client, no external services needed.
  • Flexibility: If using the Blockchain.info Wallet service, no local Bitcoin client is needed.
  • Automation: Once a sweep is setup, the system will watch for payments and initiate the sweeping.
  • Compaitibility: Any Bitcoin-qt JSON-RPC compliant wallet.

    The Windows GUI.

    The Windows GUI is under development and available to closed beta testers. Currently it supports automatic basic balance sweeping of private key to a bitcoin address. Wallet and BrainWallet sweeping are under development.

    The Ruby Library

    After we have the GUI ready to go, we will release the general library as a Ruby gem. This can be used to write your own apps, make cron jobs etc.
